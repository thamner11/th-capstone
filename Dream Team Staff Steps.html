There are a few menus: 2 main ones:

 

A Manager Menu and a User Menu

 

To access either menu the team member has to log in first. If they do not already have an account they can create an account.  

 

Once they have logged in it will register if they are either user or manager and take them one of those menus.

 

Under the Manager Menu the team member can:

 

[1] View database

 

            [A] View Users of the Dream Team

[B] Search for a certain Person

[C] View all user competencies by User ID

[D] View a report of all users and their competency levels for a given competency

[R]Return to Main Menu

 

 [2] Search for a certain Person

           

 

 [3] Add information to the database

            [A] Add a User to the Dream Team

[B] Add a new Competency

[C] Add a New Assessment to a Competency

[D] Add an Assessment result for a User for an Assessment

[R]Return to Main Menu

 

 [4] Delete an assessment result

 

 [5] Edit database

            [A] Edit a Users First Name

[B] Edit a Users Last Name

[C] Edit a Competency Date

[D] Edit a Competency Name

[E] Edit an Assessment date

[F] Edit an Assessment name

[G] Edit an Competency of the Assessment

[H] Edit an Assessment Result

[R] Return to Main Menu

 

 [6] Import assessment results

 

 [7] Export assessment results

 

 [Q] Quit
